Installation

It takes a few minutes to install Crotal.

Requirements:

Before the Installation, please check these requirements.

  • Python
  • Pip
  • Mac OS X, Linux or Unix
For Windows Users

Crotal still can not be run on windows.

Install Crotal

via pip:

$ pip install crotal

via github:

$ git clone https://github.com/dinever/crotal.git
$ cd crotal
$ sudo python setup.py install
