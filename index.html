Crotal is a static site generator.

Simpe, Static, Fast
Easy to use, and free to all.

View on Github Download Souce Code 
$ sudo pip install crotal
$ crotal init my_site && cd my_site
$ crotal generate
$ crotal server
# => Now the website is available
# => at http://localhost:8000

Crotal is Simple

Crotal is written in Python, and can be easily installed with pip, all you need to concern is the content.

Crotal is Static

Static site generated from markdown files. As is static, the site can be easily deployed on any server.

Crotal is Fast

Crotal uses incremental generation, old posts stored cached in the database, so the generation is super fast.
