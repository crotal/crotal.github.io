Create Pages

As the same as creating posts, you can create a new page by using the following command or just create it by yourself.

$ crotal new_page

Following the guide and the page will be created in the folder pages.

The page file also has a header area:

---
layout: page.html
title: "This is a new page"
date: 2013-04-20 15:10
categories: docs
slug: foo-bar
url: /foo/bar/
order: 1
---

This is basicly the same as post, however, you need to notice this changes:

  • slug: Slug here will not be used to create URL, it is just an identification.
  • url: This indicates the URL of the page. For example, the above page has the url /foo/bar/, then it will be created at /foo/bar/index.html, you can browse it on http://localhost:8000/foo/bar/.
  • order: When genearting the site, Crotal will sort the pages in their order attribute, the bigger the order, the lower it's place will be. If we didn't declare the order of a page, it will have a default order at 1000, notice that you can even declare an order with a float number, page with order 4.5 will be after the page with order 4.
