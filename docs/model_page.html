models.Page - Page¶

class crotal.models.Page(**attributes)[source]¶

Model Post for posts in the blog.

Attributes:

PATH: The relative path where this model is related to, please use the relative path to this script.

FILE_EXTENSIONS: Only the files with these file extensions shall be read by the model.

title: Title of the post.

slug: Slug of the post that may be used in its url.

pub_date: Publication date of the post.

tags: A list of Tags that the post is related to.

categories: A list of Categories that the post belongs to.

raw_content: The raw markdown content of the post.

html_content: The html format post content genearted from markdown file(raw_content).

short_html_content: A short version of the html_content, which may be represented on the index page.

create()[source]¶

Convert the page content from markdown into html format.
