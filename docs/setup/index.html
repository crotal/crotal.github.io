Configuration

After Initiating the site, you may find the configuration file named _config.yml

Settings Description Options
url URL of the site you would like to generate
title The title of the site
subtitle The subtitle of the site
author Default author of the posts
keywords Default keywords of the site
description Default description of the site
email Your email
deploy_default The deploy method you want touse rsync, git
ip If deploy by rsync, Crotal have to know your server ip.
deploy_dir If deploy by rsync, Crotal have to know the directory you want to deploy on the server.
root If publishing to a subdirectory as in http://site.com/project set 'root: /project'.
permalink See Permalinks.
theme The theme you would like to use, for further reading, see Themes.
paginate Number of the posts you would like to show on the index page.
recent_posts Number of the posts in the sidebar Recent Posts.
default_asides List of the sidebar you want to include, in the order you want them to appear.
3rd party settings Those 3rd party settings is used to show your social information on the site.
Remember the space

There must be a space between colon and value.

Reach the config value

You can use configuration value by {% config.[key] %} in tempalte files.

You can directly use the configuration file from Octopress

The configuration files between Octopres and Crotal are in common, so if you are a former Octoprss user, you can use your _config.yml file directly in Crotal.

The Default Configuration File

url: http://crotal.org
title: Crotal
subtitle: A static site generator written in Python.
author: user

keywords: mac, ios, web, octopress, hack, sql, linux, python, django
description: A static site generator which is written in Python, easy to use, and free to all.Start a blog right now using Crotal now!

email:
