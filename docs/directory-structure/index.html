Directory Structre

A basic Crotal site is like this:

▾ source/
    ▾ pages/
        docs.md
        installation.md
    ▾ posts/
        2013-04-20-hello-world.markdown
▾ public/
    ▾ _layout/
        docs.html
        sidebar.html
    ▾ archive/
        index.html
    ▾ category/
        index.html
        index.html
▸ static/
▾ themes/
    ▸ default/
  _config.yml
  db.json

The following table explains what the directories above does:

File / Directory Description
_sites After generating, the full static site will be placed in this directory.
source Folder souce includes posts and pages.
posts Posts written in markdown should be placed here.
pages Pages created should be placed here.
public In this directory, we create some custom templates, see template for more information.
static In this directory, we place some our static files, like images or videos.
themes Theme should be placed in this folder, for further reading, see Themes.
_config.yml The configuration file for the site.
db.json The database for incremental generation, including the last modify time and content of the posts & pages.
